# YouTube Tags Helper

Полезная утилита для удобной работы с тегам YouTube. Она может автоматически сортировать теги, удалять дубликаты, преобразовывать списки из разных форматов разделения.

![preview](docs/assets/preview.jpg)


## Технологии проекта

- TypeScript
- React
- Redux Toolkit
- Tailwind CSS
